# ThreadX-FOTA
Learning ThreadX, planing to implement very basic Firmware-Over-The-Air (network, eth) mechanism

## Getting Started

Clone the repository:

```
git clone https://github.com/AOSity/ThreadX-FOTA.git
cd ThreadX-FOTA
git submodule update --init --recursive
```

Build:
```
make -j
```

Flash:
```
make flash
```

Clean:
```
make clean
```
